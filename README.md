#vi
My personal cheatsheet of vi key mappings

Cut and paste stuff

key | task
-- | --
 v | select characters
 V | select line
^v | select rectangular block
-- | --
 d | cut
 y | copy ("yank to buffer")
-- | --
 P | paste before cursor
 p | paste after cursor
